# 🇰🇷 Conheça Busan - Guia Turístico

Um guia visual e informativo sobre Busan, a segunda maior cidade da Coreia do Sul. Este projeto destaca destinos imperdíveis para amantes de história, famílias e casais.

> Projeto desenvolvido como parte do aprendizado na trilha da **Rocketseat**.

## 💻 Sobre o Projeto
O site apresenta uma curadoria de templos budistas e parques históricos, utilizando uma estrutura limpa e responsiva para facilitar a leitura das informações turísticas.

## 🚀 Tecnologias Utilizadas
* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização, layout e importação de fontes externas.
* **Google Fonts:** Utilização da família de fontes *Open Sans*.

## 🏛️ Destinos Destacados
1.  **Templo Haedong Yonggungsa:** Um raríssimo templo budista à beira-mar construído em 1376.
2.  **Templo Beomeo-sa:** Um grande santuário com arquitetura preservada da Dinastia Joseon.
3.  **Parque Yongdusan:** Localizado no centro da cidade, onde fica a famosa Torre Busan.

## 🎨 Layout
O design foi pensado para ser minimalista e informativo, contendo:
- Cabeçalho chamativo.
- Seções divididas por temas (História, Locais).
- Listas de categorias (Bom para: Família, Casais, etc).
- Rodapé institucional.
